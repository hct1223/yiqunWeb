# 商业地产智慧运营解决方案图片需求

## 必需图片列表

### 1. 主图 (Hero Image)
- **文件名**: `realestate-hero.jpg`
- **尺寸**: 1200x800px 或 16:9 比例
- **内容**: 商业地产、高端商场外观或内部场景
- **建议**: 展示现代化商业综合体，体现高端、智慧运营的概念

### 2. CDP客户数据平台示意图
- **文件名**: `realestate-cdp.jpg`
- **尺寸**: 800x600px
- **内容**: CDP平台架构图或数据可视化界面
- **建议**: 展示数据整合、用户画像、标签体系的界面

### 3. MA营销自动化平台示意图
- **文件名**: `realestate-ma.jpg`
- **尺寸**: 800x600px
- **内容**: 营销自动化平台界面或流程图
- **建议**: 展示营销活动配置、自动化流程的界面

### 4. 会员分层运营示意图
- **文件名**: `realestate-membership.jpg`
- **尺寸**: 800x600px
- **内容**: 会员分层金字塔或客群分布图
- **建议**: 展示头部、腰部、泛会员三层结构

### 5. 太古汇案例图片
- **文件名**: `realestate-taikoo.jpg`
- **尺寸**: 800x600px
- **内容**: 广州太古汇实景照片
- **建议**: 展示太古汇高端商场外观或内部

### 6. 数据分析大屏
- **文件名**: `realestate-dashboard.jpg`
- **尺寸**: 1200x800px
- **内容**: 数据可视化大屏或BI报表
- **建议**: 展示会员数据、营销数据的可视化界面

## 图片规格要求
- 格式: JPG 或 PNG
- 分辨率: 至少 72 DPI
- 色彩模式: RGB
- 风格: 现代、专业、高端

## 临时占位符
在图片准备好之前，系统将使用在线占位图。
